# JSON Array Length

在 JSON 文件中显示折叠数组的长度和文件大小的 VS Code 扩展。

## 功能特点

- 自动显示折叠数组的长度
- 在文件头部显示文件大小
- 实时更新统计信息
- 文件大小阈值限制，文件过大则提醒用户是否渲染，防止影响性能

## 使用方法

1. 打开任意 JSON 文件
2. 折叠数组后会自动显示数组长度
3. 文件顶部会显示当前文件大小
